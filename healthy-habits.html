Healthy Habits

General Summary

Healthy Habits is a Spring application that allows users to go to a website, make an account, enter habits and behaviors they would like to track, and view those on a graph so they can see their progress over time.

I was looking for a realistic project I could build by myself that would combine all of what I'd been learned about programming and software. It was built with Java, Hibernate, Spring, MySQL, HTML, CSS and JavaScript.
